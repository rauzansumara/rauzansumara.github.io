## Information

This is my personal website. I used [hugo-coder theme](https://github.com/luizdepra/hugo-coder/). If you want to adopt it, I suggest you to use [original source](https://github.com/luizdepra/hugo-coder/).
1. Add the repository into your Hugo Project repository as a submodule, `git submodule add https://github.com/luizdepra/hugo-coder.git themes/hugo-coder`.
2. Configure your `config.toml`. You can either use the [this minimal configuration](https://github.com/luizdepra/hugo-coder/wiki/Configurations#complete-example) as a base, or look for a complete explanation about all configurations [here](https://github.com/luizdepra/hugo-coder/wiki/Configurations).
3. Build your site with `hugo serve` and see the result at `http://localhost:4321/`.
